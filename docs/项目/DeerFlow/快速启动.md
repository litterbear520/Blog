# 快速启动

比较有意思的是，这里通过make来管理命令的执行，我觉得特别方便，这是很值得学习的地方，或许我的项目也可以通过make来简化需要执行的命令。

## 配置

1.**克隆DeerFlow仓库**

```bash
git clone https://github.com/bytedance/deer-flow.git
cd deer-flow
```

2.**运行设置向导**

从项目根目录(deer-flow/)运行：

```bash
make setup
```
