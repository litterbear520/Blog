# 前言

1.让agent构建项目的框架，系统设计，明白项目到底在干嘛，是怎么构建的，不需要面面俱到，只需要第一个框架。

2.让ai以一个agent应用开发工程师的视角，让他告诉你如何从一到一百来完整的构建这个项目，模拟作为作者是怎么写这个项目。构建一个roadmap然后跟着一步步了解如何构建一个agent项目，在这个过程中，可以了解到某些部分为什么要这样设计，有一个根本性的就是第一性原理的认知，这个可以帮助你更深入了解agent是什么。

3.学习完一个agent项目后，着手构建自己的第一个agent，大部分都是说去GitHub找一些开源项目上面修改，但别人已经完整构建出来的一个项目其实你很难理解他为什么要这么去做，里面很多细节的设计，只能知其然却不知所以然。更适合的是去一个甲方平台：upwork，这些agent任务需求会更加清楚，找到感觉挺贴近企业要求的招工需求贴，可以交给大模型，基于这个需求让他帮助你从0到1一步一步的构建这个项目。

你构建了一个属于你的项目，你也更了解agent技术中的各种细节，能够知其然，也能知其所以然，在面试中更有自信，更加游刃有余。
